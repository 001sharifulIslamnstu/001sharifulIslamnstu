# Low-Resource NLP Research

This directory contains research projects focused on **low-resource natural language processing**, especially for **Bangla language understanding, retrieval-augmented generation, multimodal toxic speech detection, LLM-based summarization, and sentiment analysis**.

## Research Theme

Low-resource NLP aims to build reliable language technologies for languages that have limited annotated datasets, benchmarks, pretrained models, and evaluation resources.

This research direction focuses on:

- Bangla NLP
- Bangla-English code-mixed language processing
- Low-resource LLM evaluation
- Retrieval-Augmented Generation
- Toxic speech detection
- Text summarization
- Sentiment analysis
- Dataset creation and benchmarking
